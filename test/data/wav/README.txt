This folder contains audio test files to test wav input.
The following files were obtained at:
http://www-mmsp.ece.mcgill.ca/Documents/AudioFormats/WAVE/Samples.html

    M1F1-uint8-AFsp.wav     :PCM 8 kHz, 8bit unsigned, Stereo
    M1F1-int16-AFsp.wav     :PCM 8 kHz, 16bit signed, Stereo
    M1F1-int24-AFsp.wav     :PCM 8 kHz, 24bit signed, Stereo
    M1F1-int24WE-AFsp.wav   :PCM 8 kHz, 24bit signed, Stereo, WAVE_FORMAT_EXTENSIBLE
    M1F1-int32-AFsp.wav     :PCM 8 kHz, 32bit signed, Stereo
    M1F1-float32-AFsp.wav   :PCM 8 kHz, 32bit IEEEfloat, Stereo
    M1F1-float64-AFsp.wav   :PCM 8 kHz, 64bit IEEEfloat, Stereo
    6_Channel_ID.wav        :PCM 44.1 kHz, 16bit signed, 6 channels
    Pmiscck.wav             :Perverse WAVE file (9 samples) with an odd length intermediate chunk (type XxXx)

The following files were obtained at:
