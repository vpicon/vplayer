This folder contains audio test files to test wav input.
The following files were obtained at:
https://docs.espressif.com/projects/esp-adf/en/latest/design-guide/audio-samples.html
       ff-16b-1c-8000hz.mp3      : 8kHz, 16 bits, signed, mono
       ff-16b-2c-8000hz.mp3      : 8kHz, 16 bits, signed, stereo
       ff-16b-2c-44100hz.mp3     : 44.1kHz, 16 bits, signed, stereo
